# SV – Freie Waldorfschule Frankfurt

Website der Schülervertretung (SV) der Freien Waldorfschule Frankfurt am Main.

## Features

- **3D Logo** – Das SV-Logo wird als 3D-Modell (STL) in der linken oberen Ecke dargestellt und folgt dem Cursor
- **Three.js Hintergrund** – Schwebende geometrische 3D-Formen, die auf Scrollen und Mausbewegungen reagieren
- **Scroll-Animationen** – Elemente erscheinen mit eleganten Animationen beim Scrollen (GSAP + ScrollTrigger)
- **Custom Cursor** – Individueller Cursor mit Hover-Effekten
- **Responsive Design** – Optimiert für Desktop und Mobile

## Tech Stack

- [Vite](https://vitejs.dev/) – Build Tool
- [Three.js](https://threejs.org/) – 3D-Rendering
- [GSAP](https://greensock.com/gsap/) – Animationen & ScrollTrigger
